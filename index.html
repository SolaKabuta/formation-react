Découvrez notre formation React

Objectifs

Acquérir les fondamentaux de React, comprendre le cycle de vie des composants et être capable de créer des applications web interactives.

Durée

2 semaines intensives (40 heures) avec un rythme adapté aux débutants. Formation disponible en présentiel ou à distance.

Compétences

  • Création de composants
  • Gestion du state
  • Utilisation des hooks
  • Routing avec React Router

Public cible

Développeurs débutants en JavaScript, personnes en reconversion professionnelle, étudiants souhaitant se spécialiser en frontend.

Programme détaillé

Semaine 1 : Les fondamentaux

Introduction à React

Présentation de l'écosystème React, JSX, création du premier composant.

Composants et props

Découpage en composants réutilisables, passage de données via les props.

State et événements

Gestion de l'état local, gestion des événements utilisateur.

Semaine 2 : Approfondissement

Les hooks React

useState, useEffect, useContext et création de hooks personnalisés.

Gestion des formulaires

Contrôlé vs non-contrôlé, validation, soumission.

React Router

Navigation entre vues, paramètres d'URL, redirections.

Projet final

Mise en pratique sur un cas concret avec accompagnement personnalisé.
